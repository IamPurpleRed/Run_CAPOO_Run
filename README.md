Run CAPOO Run !!
===
###### tags:`Github`, `project`, `OOP`

## Description
109-1 物件導向程式設計 期末專題

## How to Play
玩家需要透過滑鼠左鍵右鍵操控角色移動，閃避迎面而來的障礙物，而隨著時間過去障礙物的速度會越來越快，努力讓珈波拿到最高分吧！
